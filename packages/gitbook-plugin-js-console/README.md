# gitbook-plugin-js-console

GitBook plugin that provide interactive JavaScript console.

## Install

Install with [npm](https://www.npmjs.com/):

    npm install gitbook-plugin-js-console

## Usage

put "js-console" to your `book.json`

```json
{
  "gitbook": ">=3.0.0",
  "plugins": [
    "js-console"
  ]
}
```

See [example](./example/)
